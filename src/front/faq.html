Was bedeutet System.out.printf in Java?

Es handelt sich um die Ausgabe von formatiertem Text auf der Konsole

Wie implementiert man eine For-Schleife in Java?

Mit drei Parametern in Klammern: for (int i = 0; i < j; i++). i ist hierbei die Variable, die hochgezählt (und am Anfang auf 0 gesetzt) wird bis j erreicht wird. i++ gibt an, dass in Einerschritten hochgezählt wird, kann aber beliebig variiert werden.
